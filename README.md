# StudentDeals - E-commerce Aggregator for Students

A modern web application that helps students find the best deals across multiple e-commerce websites by searching for products and filtering by budget range.

## Features

- **Product Search**: Search for any product across multiple e-commerce sites
- **Budget Filtering**: Set minimum and maximum budget ranges
- **Multi-Platform Results**: Results from Flipkart, Amazon, Myntra, and Snapdeal
- **Price Comparison**: Compare prices across different platforms
- **Direct Purchase Links**: Click to go directly to the original e-commerce site
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Modern UI**: Beautiful gradient design with smooth animations

## How to Use

1. **Open the Website**: Simply open `index.html` in your web browser
2. **Search for Products**: 
   - Enter your search term (e.g., "white sneakers", "laptop", "smartphone")
   - Set your budget range (minimum and maximum price)
   - Click the Search button
3. **Browse Results**: 
   - View products from different e-commerce sites
   - Sort by price, rating, or site
   - Click on any product card or "Buy Now" button to visit the original site
4. **Make Purchase**: Complete your purchase on the original e-commerce platform

Try these search terms to see the demo in action:

## Technical Details

- **Frontend**: Pure HTML5, CSS3, and JavaScript (ES6+)
- **Styling**: Modern CSS with gradients, animations, and responsive design
- **Icons**: Font Awesome for beautiful icons
- **Data**: Mock data simulating real e-commerce products
- **Browser Support**: Works on all modern browsers

## Future Enhancements

This is a demo version with mock data. For a production version, you would need:

1. **Real API Integration**: Connect to actual e-commerce APIs
2. **Database**: Store product information and user preferences
3. **Backend**: Node.js, Python, or PHP backend for API calls
4. **Authentication**: User accounts and wishlists
5. **Price Tracking**: Monitor price changes over time
6. **Reviews Integration**: Show actual user reviews
7. **Image Loading**: Real product images from e-commerce sites
